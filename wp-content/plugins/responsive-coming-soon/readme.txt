== Description ==

Responsive Coming Soon plugin is  provide you creative and well-crafted Coming soon / Under construction template. It is built on Bootstrap3 Framework and is based on latest HTML5 for a robust and fast site. Its interesting design is perfect for any site type.

It features a full screen slide show and multi-page content from one screen. The look stands out and will remain memorable to all of your visitors.

== Installation ==

1. Upload the entire `responsive-coming-soon` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Responsive Coming Soon Panels menu to customize this plugin.
4. Finally Enjoy it.
